# dotfiles

. files

## Install

```bash
git clone https://github.com/Hurtak/dotfiles.git __dotfiles && \
chmod u+x __dotfiles/install.sh && \
./__dotfiles/install.sh && \
rm -rvI __dotfiles
```

## Finetune configs

```bash
vim ~/.bashrc && \
vim ~/.bash_aliases && \
vim ~/.gitconfig && \
vim ~/.vimrc
```
