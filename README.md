# dotfiles

. files

## Install

### Linux

```bash
git clone https://github.com/Hurtak/dotfiles.git __dotfiles && \
chmod u+x __dotfiles/linux.sh && \
./__dotfiles/linux.sh && \
rm -rfv __dotfiles && \
vim ~/.bashrc && \
vim ~/.bash_aliases && \
vim ~/.gitconfig && \
vim ~/.vimrc
```

### Mac

```bash
git clone https://github.com/Hurtak/dotfiles.git __dotfiles && \
chmod u+x __dotfiles/mac.sh && \
./__dotfiles/mac.sh && \
rm -rfv __dotfiles && \
vim ~/.bash_profile && \
vim ~/.gitconfig && \
vim ~/.vimrc
```
